

# Online Retail Sales Analytics Dashboard

Developed an end-to-end Business Intelligence solution using Python and Power BI to analyze online retail sales data.

## Key Features

- Data cleaning and preprocessing using Python
- Interactive Power BI dashboard with 8 analytical pages
- KPI tracking and business performance monitoring
- Product performance and returns analysis
- Customer segmentation and behavior analysis
- Logistics and shipping efficiency insights
- Profitability and cost analysis
- Revenue trend analysis and forecasting

## Tools Used

- Python (Pandas)
- Google Colab
- Power BI
- DAX
- CSV Dataset

## Dashboard Modules

1. Business Performance Overview
2. Product Analysis
3. Customer & Regional Insights
4. Logistics & Shipping Analysis
5. Profitability Analysis
6. Customer Segmentation
7. Revenue Trend Analysis
8. Forecasting & Advanced Insights
